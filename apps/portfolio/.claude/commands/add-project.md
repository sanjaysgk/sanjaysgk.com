description: Add a new project to the portfolio
---

Add a new project entry to the portfolio. Follow these steps:

1. Read `src/data/resume.tsx` to understand the existing project structure
2. Ask the user for project details if not provided: title, description, technologies, dates, links (GitHub, live demo), and whether it's a "Product" or "Research" project
3. Add the project to the appropriate array in `resume.tsx` following the existing pattern
4. If the project needs a logo/image, note that it should be added to `public/`
5. Verify the project appears correctly by running `npm run build`

Project details: $ARGUMENTS
